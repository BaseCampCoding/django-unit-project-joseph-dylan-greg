print()

The print function displays the string value in side its parentheses on the screen.

if statements

The most common type of flow control that executes the block of code if the condition is True. The clause is skipped if False.

else statements

An if statement can optionally be followed by an else statement. The else statement is executed only when the if statement's condition is False.

elif statements

An if statement can optionally be followed by an elif statement if the case arrives you want more than one possibility. The elif statement is always followed by an if statement or another elif statement.

Variable

A variable is initialized or created the first time a value is stored inside of it.

len()

The len function is passed onto a string value or a variable containing a string, and the function evaluates the integer value of the number of characters within the string.

Local Scope

Parameters and variables that are assigned in a called function.

Global Scope

Variables that are assigned outside of all functions are considered a global scope.

List

A list is a value that contains multiple values in an ordered sequence.

input()

The input function waits for the user to type some text on the keyboard and press enter.

Boolean Value

A data type that only has two values: True and False.

Escape Characters

An escape character lets you use characters that are otherwise impossible to put into a string. An escape character consists of a backslash followed by the character you want to add to the string.

join()

The join method is useful when you have a list of strings that need to be joined together.
